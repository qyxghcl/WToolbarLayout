# WToolbarLayout
An adapter for status bar

## how to use
## Step1
Add it in your root build.gradle at the end of repositories:
```
allprojects {
   repositories {
   ...
   maven { url 'https://jitpack.io' }
   }
}
```

## Step2
 Add the dependency
```
implementation 'com.github.qyxghcl:WToolbarLayout:v1.0.0'
```
